## List Of Hooks

| Name                                                                                      | Description                                                                                                                                                                   |
| ----------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [useAnimatePresence](https://malkiii.github.io/realtime-hooks/guide/useAnimatePresence)   | Animate an element when it enters or leaves the DOM.                                                                                                                          |
| [useArray](https://malkiii.github.io/realtime-hooks/guide/useArray)                       | Returns an array with common array methods.                                                                                                                                   |
| [useAsync](https://malkiii.github.io/realtime-hooks/guide/useAsync)                       | Execut an `async` function and returns a loading state, return data and other values.                                                                                         |
| [useAudio](https://malkiii.github.io/realtime-hooks/guide/useAudio)                       | Returns an audio `ref` as well as its state and controler methods.                                                                                                            |
| [useAutoComplete](https://malkiii.github.io/realtime-hooks/guide/useAutoComplete)         | Returns the result array of a search text.                                                                                                                                    |
| [useClickAway](https://malkiii.github.io/realtime-hooks/guide/useClickAway)               | Execute a function when clicking outside an element.                                                                                                                          |
| [useClipboard](https://malkiii.github.io/realtime-hooks/guide/useClipboard)               | Copy to clipboard and return a temporary `copied` state.                                                                                                                      |
| [useClock](https://malkiii.github.io/realtime-hooks/guide/useClock)                       | Returns a date in a duration starting from an initial date or return the current date.                                                                                        |
| [useColorScheme](https://malkiii.github.io/realtime-hooks/guide/useColorScheme)           | Returns the current system color scheme (`dark` or `light`).                                                                                                                  |
| [useCounter](https://malkiii.github.io/realtime-hooks/guide/useCounter)                   | Simple counter with `step`, `min` and `max` value.                                                                                                                            |
| [useCss](https://malkiii.github.io/realtime-hooks/guide/useCss)                           | Insert some actual CSS using a `CSS Object`.                                                                                                                                  |
| [useCursorType](https://malkiii.github.io/realtime-hooks/guide/useCursorType)             | Track the cursor type (`pointer`, `text`, `not-allowed`...)                                                                                                                   |
| [useDebouncedState](https://malkiii.github.io/realtime-hooks/guide/useDebouncedState)     | Update the state value after a delay.                                                                                                                                         |
| [useDevice](https://malkiii.github.io/realtime-hooks/guide/useDevice)                     | Returns the current device type (`desktop`, `mobile`, `smartTV`...)                                                                                                           |
| [useEasing](https://malkiii.github.io/realtime-hooks/guide/useEasing)                     | Change a value from `start` to `end` using an [`animation timing function`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation-timing-function).                      |
| [useEventListener](https://malkiii.github.io/realtime-hooks/guide/useEventListener)       | Add one or multiple event listeners.                                                                                                                                          |
| [useFetch](https://malkiii.github.io/realtime-hooks/guide/useFetch)                       | Fetch data using a URL and search queries...                                                                                                                                  |
| [useFileDropzone](https://malkiii.github.io/realtime-hooks/guide/useFileDropzone)         | Handles file dropping to a label of a file input.                                                                                                                             |
| [useGeolocation](https://malkiii.github.io/realtime-hooks/guide/useGeolocation)           | Track the current geo location of the client.                                                                                                                                 |
| [useHover](https://malkiii.github.io/realtime-hooks/guide/useHover)                       | Returns a `hovered` state, and it can use a `delay` value as well.                                                                                                            |
| [useImageLoading](https://malkiii.github.io/realtime-hooks/guide/useImageLoading)         | Loads an image using its `URL` and returns the loading state and the loaded `image`.                                                                                          |
| [useIntersection](https://malkiii.github.io/realtime-hooks/guide/useIntersection)         | Handles `IntersectionObserver` and returns the intersecting state (the element is in the view or not).                                                                        |
| [useInterval](https://malkiii.github.io/realtime-hooks/guide/useInterval)                 | Uses `setInterval` and returns `start` and `stop` methods...                                                                                                                  |
| [useIsomorphicEffect](https://malkiii.github.io/realtime-hooks/guide/useIsomorphicEffect) | Uses `useEffect` on the client and `useLayoutEffect` on the server.                                                                                                           |
| [useKeyboard](https://malkiii.github.io/realtime-hooks/guide/useKeyboard)                 | Bind keyboard `keys` or `hotkeys` with handlers.                                                                                                                              |
| [useLocalStorage](https://malkiii.github.io/realtime-hooks/guide/useStorage)              | Get and Set a specific value in the local storage.                                                                                                                            |
| [useMap](https://malkiii.github.io/realtime-hooks/guide/useMap)                           | Returns an object with common map methods such as `get`, `set`, `has`, `delete`...                                                                                            |
| [useMount](https://malkiii.github.io/realtime-hooks/guide/useMount)                       | Execute a callback after the component is mounted.                                                                                                                            |
| [useMouse](https://malkiii.github.io/realtime-hooks/guide/useMouse)                       | Track the mouse position and state (down or up, in or out).                                                                                                                   |
| [useMutation](https://malkiii.github.io/realtime-hooks/guide/useMutation)                 | Handles `MutationObserver` and returns the current mutations (changes being made to the DOM tree).                                                                            |
| [useOnlineState](https://malkiii.github.io/realtime-hooks/guide/useOnlineState)           | Returns whether the user is online or not.                                                                                                                                    |
| [usePageVisibility](https://malkiii.github.io/realtime-hooks/guide/usePageVisibility)     | Returns the tab visible state (the tab contents is visible or hidden).                                                                                                        |
| [useScroll](https://malkiii.github.io/realtime-hooks/guide/useScroll)                     | Track the scroll `position` ,`progress` and `direction`.                                                                                                                      |
| [useScrollEnd](https://malkiii.github.io/realtime-hooks/guide/useScrollEnd)               | Execute a callback when the scrolling is close to the end with a specific `offset`.                                                                                           |
| [useSelection](https://malkiii.github.io/realtime-hooks/guide/useSelection)               | Returns the current selected text as well as the target element and its rectangle...                                                                                          |
| [useSessionStorage](https://malkiii.github.io/realtime-hooks/guide/useStorage)            | Get and Set a specific value in the session storage.                                                                                                                          |
| [useSet](https://malkiii.github.io/realtime-hooks/guide/useSet)                           | Returns a set with common set methods.                                                                                                                                        |
| [useSize](https://malkiii.github.io/realtime-hooks/guide/useSize)                         | Track the element size using `ResizeObserver`.                                                                                                                                |
| [useSpeech](https://malkiii.github.io/realtime-hooks/guide/useSpeech)                     | Returns a text speaker with available voices as well as its state and controler methods.                                                                                      |
| [useStateHistory](https://malkiii.github.io/realtime-hooks/guide/useStateHistory)         | Stores a specific number of the previous state values.                                                                                                                        |
| [useStateStatus](https://malkiii.github.io/realtime-hooks/guide/useStateStatus)           | Returns the state and its current `status` that is returned by a handler.                                                                                                     |
| [useSwipeAction](https://malkiii.github.io/realtime-hooks/guide/useSwipeAction)           | Handle the user swipe actions, and it can also handle the mouse swipes.                                                                                                       |
| [useTimeout](https://malkiii.github.io/realtime-hooks/guide/useTimeout)                   | Uses `setTimout` and returns `start` and `stop` methods...                                                                                                                    |
| [useToggle](https://malkiii.github.io/realtime-hooks/guide/useToggle)                     | Toggles the state between given values.                                                                                                                                       |
| [useUnmount](https://malkiii.github.io/realtime-hooks/guide/useUnmount)                   | Execute a callback before the component is unmounted.                                                                                                                         |
| [useUpdateEffect](https://malkiii.github.io/realtime-hooks/guide/useUpdateEffect)         | Ignore the first render of the component.                                                                                                                                     |
| [useVideo](https://malkiii.github.io/realtime-hooks/guide/useVideo)                       | Returns a video `ref` as well as its state and controler methods.                                                                                                             |
| [useViewport](https://malkiii.github.io/realtime-hooks/guide/useViewport)                 | Track the screen `viewport` (width and height) and `orientation`.                                                                                                             |
| [useWebRTC](https://malkiii.github.io/realtime-hooks/guide/useWebRTC)                     | Uses the user media devices (`camera` and `microphone`) and `screen capture` and returns their states and controls as well as a `recorder` that records their current stream. |
